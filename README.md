# AndroidTV-Mouse-Navigator

Android TV kumandasıyla çalışan sistem geneli bir mouse imleci sağlayan örnek Android TV uygulaması.

## Özellikler
- Foreground servis ile sürekli çalışan overlay cursor
- DPAD yön tuşlarıyla yumuşak ve sınır kontrollü hareket
- OK/ENTER tuşuyla MotionEvent tabanlı dokunma simülasyonu
- Back tuşuyla servisi durdurma ve uygulamadan çıkış
- Compose ile çizilen yüzen cursor

## Başlangıç
1. Android Studio ile projeyi açın.
2. Bir Android TV emülatörü veya gerçek cihaz bağlayın.
3. Uygulamayı derleyip çalıştırın.
4. Uygulama içindeki "Servisi Başlat" butonuna basın ve overlay iznini verin.
5. Accessibility servis ayarlarından TvMouseNavigator servisini etkinleştirin.
